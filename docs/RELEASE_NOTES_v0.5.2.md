## Wonder v0.5.2

Wonder 是一款面向 macOS 和 Windows 的后台划词翻译工具。

### 本次更新

- 支持在兼容应用中选中文本后显示 Wonder 翻译气泡。
- macOS 菜单栏快捷翻译支持输入后自动翻译。
- 优化液态玻璃翻译窗口和顶部拖动条。
- 原文与译文区域均支持一键复制。
- 翻译窗口支持拖动，并可点击窗口外自动关闭。
- 默认使用无需 API Key 的 MyMemory 免费翻译服务。

### 下载说明

- `Wonder-v0.5.2-macOS-arm64.dmg`：Apple Silicon Mac 安装镜像。
- `Wonder-v0.5.2-macOS-arm64.zip`：Apple Silicon Mac 应用压缩包。
- `Wonder-v0.5.2-windows-x64.exe`：64 位 Windows 自包含单文件程序。

### 首次打开

macOS 版本尚未经过 Apple 公证，可能需要前往“系统设置 → 隐私与安全性”选择“仍要打开”。Windows 版本尚未使用商业代码签名证书，SmartScreen 可能显示“未知发布者”。

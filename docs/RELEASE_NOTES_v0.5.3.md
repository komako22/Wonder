## Wonder v0.5.3

本次版本重点修复全局选区监听误触发问题，让 Wonder 只在用户真实划词时显示翻译气泡。

### 修复内容

- 修复 macOS 后台运行时，按住 Shift 再按其他按键会错误模拟一次 `⌘C` 的问题。
- 移除会误触发剪贴板读取的全局 Shift 键抬起监听。
- 普通单击和桌面图标双击不再触发剪贴板兜底或翻译气泡。
- 只有鼠标形成有效拖动选区后，才允许使用安全复制作为兼容兜底。
- 双击选择文本时只读取系统辅助功能选区，不主动模拟复制。
- Windows 同步增加最小拖动距离判断，减少普通点击和窗口切换造成的误触发。
- 修复本地 macOS 签名脚本对多行钥匙串路径处理不安全的问题。

### 下载说明

- `Wonder-v0.5.3-macOS-arm64.dmg`：Apple Silicon Mac 安装镜像。
- `Wonder-v0.5.3-macOS-arm64.zip`：Apple Silicon Mac 应用压缩包。
- `Wonder-v0.5.3-windows-x64.exe`：64 位 Windows 自包含单文件程序。

### 首次打开

macOS 版本尚未经过 Apple 公证，可能需要前往“系统设置 → 隐私与安全性”选择“仍要打开”。Windows 版本尚未使用商业代码签名证书，SmartScreen 可能显示“未知发布者”。
